# SigmaZ

Windows个人电脑通用微处理器测试。

The Generic Microsoft(R) Windows(R) PC Performance Evaluation Utility.

## 主要功能

*   **多架构支持：** 原生支持 16 位 (Win16)、32 位 (Win32) 和 64 位 (Win64) Windows 环境。
*   **测试项目：** 涵盖整数、浮点、内存、加密、压缩、矩阵运算。
*   **超时机制：** 整数、浮点、加密、压缩、矩阵测试限时 60 秒并支持超时结算；内存测试使用固定 3 秒窗口测量带宽。
*   **评分基准：** 以 Intel 486 DX2-66（估算值） 为 100 分。
