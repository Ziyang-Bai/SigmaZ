# 快速开始

## 1. 选择版本

SigmaZ 提供三个版本以适配不同的操作系统环境：

*   `sigma16.exe` (Win16)：适用于 Windows 3.1, Windows 9x 的实模式环境。建议用于 286/386/486 等早期硬件。
*   `sigma32.exe` (Win32)：适用于 Windows 95 至 Windows 11。支持多线程并发计算。
*   `sigma64.exe` (Win64)：适用于 Windows 7/10/11 等现代 64 位操作系统，支持更大的内存寻址和新的指令集优化。

## 2. 运行测试

程序启动后，您可以通过以下步骤进行完整的性能评估：

1.  在主界面选择 **All** 标签页。
2.  点击 **Start All Tests** 按钮。
3.  程序将依次执行所有基准测试项目。

> **注意：** 在性能较低的硬件上（如 386/486），测试过程中界面可能会出现短暂的“无响应”状态，这是由于单线程计算占用了 CPU 资源。请耐心等待，测试完成后界面会自动恢复响应。

## 3. 免责声明

本工具会对您的硬件 (`CPU` 和 `内存`) 施加巨大的压力。虽然已内置保护机制，但在不稳定或超频系统上运行带来的风险请自行承担。
